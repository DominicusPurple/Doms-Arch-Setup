**Bash** script that recreates my ArchLinux setup in minutes.

Open your prefered terminal emulator and enter the following commands.

* `git clone https://github.com/DominicusPurple/Doms-Arch-Setup`

* `cd Doms-Arch-Setup`

* `./install.sh`
